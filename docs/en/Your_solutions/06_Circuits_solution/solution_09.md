## 9. Current

**Problem:** Charge flowing through the wire is given by the function of time $Q(t) = 5t^2+5$. What is the current at $t=3\text{ s}$?

### Physical Concept
Electric current ($I$) is defined as the rate at which electric charge ($Q$) flows through a given area over time. Mathematically, the instantaneous current is the first derivative of the charge function with respect to time:
$$I(t) = \frac{dQ(t)}{dt}$$

### Step 1: Find the Current Function
We need to take the derivative of the given charge function $Q(t) = 5t^2 + 5$ with respect to time ($t$).
Using the standard power rule for derivatives in calculus:
$$I(t) = \frac{d}{dt}(5t^2 + 5)$$
$$I(t) = (2 \cdot 5t^{2-1}) + 0$$
$$I(t) = 10t$$

This new equation, $I(t) = 10t$, gives us the formula for the instantaneous current at any given second.

### Step 2: Calculate the Current at $t = 3\text{ s}$
Now, to find the specific current at $3\text{ seconds}$, we simply substitute $t = 3$ into our derived current function:
$$I(3) = 10 \cdot (3)$$
$$I(3) = 30\text{ A}$$

---

### Final Conclusion
The instantaneous current flowing through the wire at exactly $3\text{ seconds}$ is **$30\text{ A}$** (Amperes).
